## Install

Create and activate a virtualenv
```
virtualenv env
env\Scripts\activate
```

Install libraries
```
pip install -r requirements.txt
```

## Usage

Try the file 
```
Dendrogram Exploitation - agg_clust_dataset_test_mean_std.py
```
