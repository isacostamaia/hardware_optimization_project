# Prediciton

## Requirements

* [Python 3](https://www.python.org/)
* [virtualenv](https://pypi.org/project/virtualenv/)

## Install

Create and activate a virtualenv
```
virtualenv env
env\Scripts\activate
```

Install libraries
```
pip install -r requirements.txt
```


## Usage

cd Modeling
Then run program, and check results from output directory specified in settings
```
python -m Regression



```
